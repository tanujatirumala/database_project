To run create.sql file please run the command in the psql: \i 'path of create.sql file' -- This command will create the tables present in the script file.
To run load.py please run the command: python3 load.py -- It will ask for dat file. Please enter the data file to load the data into tables.
To run the query please run the command : python3 query.py -- It will ask for council name and troop number. Please enter those values to get the results.
To drop schema please run the command : DROP SCHEMA Z1947594 CASCADE;
